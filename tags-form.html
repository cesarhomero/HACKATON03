ETIQUETAS DE FORMULARIO

form

Representa un formulario, consistiendo de controles que puede ser enviado a un servidor para el procesamiento de los datos ingresados

label

Representa el título de un control de formulario

input

Representa un campo de datos escrito que permite al usuario modificarlos

select

Representa un control que permite la selección entre un conjunto de opciones

option

representa una opción dentro de un elemento "select"

button

Representa un botón

textarea

Representa un área o espacio donde se puede escribir texto en un determinado número de carácteres
